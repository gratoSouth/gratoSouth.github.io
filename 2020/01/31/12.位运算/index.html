位运算介绍
程序中所有数在计算机内存中都是以二进制的形式存储的。位运算说穿了，就是直接对整数在内存中的二进制位进行操作。由于位运算直接对内存数据进行操作，不需要转成十进制，因此处理速度非常快。

符号：

  • &与、|或、^异或、~取反
  • <<左移（高位丢弃，低位补零、>>右移（对无符号数，高位补零，有符号数，各编译器处理不一，有的补符号位（算术右移），有的补零（逻辑右移））
  • XOR-异或：相同为0，不同为1，也可用不进位加法来理解。
    特点：
    x ^ 0 = x
    x ^ 1s = x
    x ^ (    x) = 1s
    x ^ x = 0
    a ^ b = c
    a ^ b = c、a ^ c = b, b ^ c = a
    a ^ b ^ c = a ^ (b ^ c) = (a ^ b) ^ c

实战常用的位运算操作
x & 1 == 1 or == 0 判断奇偶（x % 2 == 1）
x = x & (x - 1) 清零最低位的1
x & -x(取反再加1) 得到最低位的1在的位置
