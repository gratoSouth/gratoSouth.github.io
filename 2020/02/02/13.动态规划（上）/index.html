动态规划 Dynamic Programming

programming(递推的意思)

要点

  1. 递归+记忆化 ->递推
  2. 状态的定义：opt[n], dp[n], fib[n]
    （例一就为一个一维数组，例二为二维数组）
  3. 状态转移方程：opt[n] = best_of(opt[n-1], opt[n-2], …)
    （可能会出现if else如例二）
  4. 最优子结构
    （在解决原问题的同时，还解决了很多最优子问题）

DP vs 回溯 vs 贪心

  • 回溯（递归）- 重复计算
  • 贪心 - 永远局部最优
  • DP - 记录局部最优子结构/多种记录值

70.爬楼梯

就是斐波拉契数列问题

1.回溯的方法
f(n) = f(n-1) + f(n-2), f(0) = 0, f(1) = 1, f(2) = 2
即运用Recursion + Memrization的方法解决

2.DP的方式

解动态规划问题的最重要的两点

  1. DP状态的定义
    f(n):到第n阶的总走法个数
  2. DP方程
    f(n) = f(n-1) + f(n-2)

120.三角形最小路径和

1.递归 o(2^n)
超时

2.贪心 不能做

3.动态规划 o(m*n)

  1. DP状态的定义
    DP[i, j]: bottom->(i,j)path sum, min
  2. DP方程
    DP[i, j]: min(DP(i+1, j), DP(i+1, j+1)) + Triangle[i, j]
  3. 初始状态
    DP[m-1, j] = Triangle[m-1, j]
